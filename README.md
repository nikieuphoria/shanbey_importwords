# shanbey_importwords
一个 Chrome 插件，用于将扇贝单词中的单词本导出为本地 JSON 文件，支持自动翻页。
功能

✅ 自动抓取扇贝单词列表，支持多页翻页
✅ 导出为 .json 文件
⬜ PTE Write From Dictation 爬取 —— 计划中，暂未实现

安装方法

下载或克隆本仓库
打开 Chrome，访问 chrome://extensions
开启右上角的开发者模式
点击加载已解压的扩展程序，选择项目文件夹（即包含 manifest.json 的那层目录）

使用方法

登录 web.shanbay.com，进入你的单词本页面
点击浏览器工具栏中的插件图标
点击开始爬取
插件会自动翻页，完成后下载 scallop_word_list.json 文件
